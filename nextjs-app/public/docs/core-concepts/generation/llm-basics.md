# LLM Basics

Large Language Models are the foundation of modern AI applications.

## Key Concepts

- **Tokens**: Basic unit of text (roughly 4 characters)
- **Context Window**: Maximum tokens the model can process (varies by model)
- **Temperature**: Controls randomness in outputs (0 = deterministic, 1 = creative)
- **Top-p Sampling**: Controls diversity of outputs

## Popular LLMs

### OpenAI Models

- **GPT-4**: State-of-the-art performance
- **GPT-3.5-Turbo**: Fast and cost-effective

### Anthropic

- **Claude 3**: Strong reasoning capabilities

### Open Source

- **Llama 2**: Meta's open model
- **Mistral**: Fast and efficient

## Parameters vs. Fine-tuning

- **Parameters**: Model inputs that control behavior
- **Fine-tuning**: Updating model weights for specific tasks
