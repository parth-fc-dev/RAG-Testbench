# Why Use RAG?

RAG addresses several critical challenges in AI systems:

## The Problem with Standard LLMs

- **Knowledge Cutoff**: LLMs have a training cutoff date
- **Hallucinations**: May generate plausible-sounding but false information
- **No Context**: Cannot access domain-specific or proprietary data
- **Static Knowledge**: Requires retraining to update information

## RAG Solution

RAG solves these issues by:

- Accessing real-time information
- Grounding responses in actual documents
- Supporting domain-specific knowledge
- Maintaining cite-able sources

## When to Use RAG

- Document-heavy applications (customer support, documentation)
- Real-time data requirements (news, analytics)
- Fact-based Q&A systems
- Compliance and regulated industries
